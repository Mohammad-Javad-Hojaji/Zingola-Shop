<h2 id="english">🚀 About the Project</h2>

### The purpose of the project 

to create the Zingula Shop website was an idea that came to my mind suddenly and I wanted to test my skills in the field of website design by creating this project. When I started the project, I was always excited to continue working, but it wasn't always like that and sometimes when I encountered many failures, I would leave my computer in despair.

### Key Learnings

* **Component-Based Architecture:** Learning to break down a UI into reusable, maintainable components.
* **State Management:** Handling shopping cart logic, product filtering, and category navigation using React Hooks.
*   **Responsive Design:** Ensuring the shop looks great on both desktops and mobile devices. 
*  **Lifecycle & Hooks:** Gaining practical experience with `useState` and `useEffect`.
<h2 id="Installation">🛠 Installation & Usage</h2>

Follow these step to get the project up and running on your local machine:

### 1. Prerequisites
Make sure you have [Node.js](https://nodejs.org/) installed on your system.

### 2. Clone the Repository
First, clone this repository to your local machine:
```bash
git clone https://github.com/Mohammad-Javad-Hojaji/Zingola-Shop.git

```

### 3. Install Dependencies

Navigate into the project directory and install the necessary packages:
                                                  
```
cd Zingola-Shop

npm install
```

### 4. Start the Application

Run the development server to see the shop in action:
                       
```
npm start
```

The app will automatically open in your default browser at [http://localhost:3000](http://localhost:3000).

<h3>5. Database implementation</h3>

Because our site is built using the Fake API, you must run the following command in a cmd or other terminal in the same directory (folder) to avoid problems with the site process.
  
 ```
npx json-server --watch data/db.json --port 8000
```

<h2 id="Documentation">📖 Documentation & Scripts</h2>

In the project directory, you can run:

### `npm start`

Runs the app in the development mode.

### `npm test`

Launches the test runner in the interactive watch mode.

### `npm run build`

Builds the app for production to the `build` folder.

### `npx json-server --watch data/db.json --port 8000`
